# Dotfiles

## Installation
```bash
git clone git@github.com:rafaelderolez/dotfiles.git .dotfiles && cd .dotfiles && ./install
```

## Features
1. Set up SSH key
2. Install homebrew
3. Install global node modules
4. Set Dock apps & settings
5. Set macos default apps
